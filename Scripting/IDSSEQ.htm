SEQ.IDS

This file only appears in TOB. The sequences may be related to some of the new commands which appear in TOB's ACTION.IDS.

For some reason they make me think of the commands in PST which played an animation sequence. 

After testing these commands, I have found that these SEQ stats work in exactly the same fashion as the SetAnimState(O:Object*,I:Anim*AnimStat) command in PST when used in SetSequence(I:Sequence*SEQ).

It allows you to simulate various states without using a spell to induce the state. 

KR

345 SetSequence(I:Sequence*SEQ)
