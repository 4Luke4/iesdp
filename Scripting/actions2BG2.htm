215 FollowObjectFormation(Object*,I:Formation*,I:Position*)

You can use this command to back stab, because it will enable him to move behind them first.

FollowObjectFormation(LastseenBy(Myself),4,4)

233 CreateCreatureObjectOffScreen(S:ResRef*,O:Object*,I:Usage1*,I:Usage2*,I:Usage3*) 

this will basically creature the creature just off the edge of the visible screen as determined by the object.

252 CreateCreatureObjectOffset(S:ResRef*,O:Object*,P:Offset*)


this allows you to creature a creature realtive to the target at the set offset.
so

CreateCreatureObjectOffset("hello",Player1,[0.10])

would create a creature 10 game distance units in front of player 1
