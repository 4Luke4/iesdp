New Additional actions for Planescape : Torment

by LordRampage and Platter.

Simply copy and paste these three into your ACTION.IDS file using NearInfinity.  If you don't have NearInfinity, or your computer cannot run java scripts, use Infinity Explorer, load up PS:T, export the ACTION.IDS file to your desktop, open it up with notepad, cut & paste, then save the file and move it to your PS:T override folder.

The commands:

170 RevealAreaOnMap(S:ResRef*)
173 AddJournalEntry(I:Entry*)
176 EscapeAreaDestroy(I:Delay*)
182 MakeUnselectable(I:Time*)
188 TakePartyItemAll(S:Item*)
190 FinalSave()


They work exactly like their Baldur's Gate conterparts. For those that don't know what they do:

RevealAreaOnMap shows a place on the map using the map entry's refferrence number.
Ex: RevealAreaOnMap("AR1200") shows the Fortress of Regrets on your map.

AddJournalEntry adds an entry based upon the string used.
Ex: AddJournalEntry(47938) adds the text "Leave" to your journal as an entry.

EscapeAreaDestroy is a variant of EscapeArea. That's all I know.

MakeUnselectable(I:Time*) makes the party member unselectable for the specified amount of time.

TakePartyItemAll(S:Item*) takes all of the specified item (ex- even if you have more than one) from your party and gives them to the creature performing the action.

FinalSave makes a saved game file called "Final Save".  Perfect for final battles.  It also exports The Nameless One as a CHR file to a folder called "Characters", but that's the only side affect.
