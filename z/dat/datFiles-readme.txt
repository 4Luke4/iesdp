+) Section 1. About
===================
  These archive contains replacement .dat files, specifying the effects for Baldurs Gate I: Tales of the Sword Coast, Baldurs Gate II: Throne of Bhaal, Icewind Dale 1, Icewind Dale 2 and Planescape: Torment. The files contain new information, clarifications as well as spelling and grammar corrections.
  If an effect does not work as described, post details at the IESDP update board:
    http://forums.gibberlings3.net/index.php?showforum=54

+) Section 2. (Un)Installation
==============================
Install
  DLTCEP:
    Replace the existing .dat files in the DLTCEP directory with the provided files.

  IEEP:
    Place the provoded files in the \Configuration directory, renaming them to match the existing files (e.g. rename BG2Effects.dat to ToBEffects.dat).

NB. You may with to back up any files you are replacing.

Uninstall
  To uninstall, simply overwrite the file you replaced with the back-up you made.

+) Section 3. Usage
===================
Once the file is installed, the new effect descriptions should appear within DLTCEP/IEEP.
