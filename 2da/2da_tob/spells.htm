spells.2da

This file defines which spells are given to priest-types on level-up. Priests gain spells SPPR[1-9][**] where ** is the column from this file, i.e. with the default file, priests will gain all SPPRxYY spells, where x is 1-7 and YY is 50 or below.
This file seems to have no effect on mages.
In BG2 ToB we have:
