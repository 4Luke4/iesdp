Lists of 2DA files according to the given Infinity Engine game

 

  • Baldur's Gate

  • Baldur's Gate Tales of the Sword Coast

  • Icewind Dale

  • IWD Heart of Winter

  • IWD Trials of Luremaster

 

For Baldur's Gate :

2DA file name

 Short description

ABCLASRQ.2DA

  minimum stats (Str,Dex etc.) per class/kit

ABDCDSRQ.2DA

  Minimum stat required to dual-class with this class

ABDCSCRQ.2DA

  Minimum stat required to be able to dual-class

ABDUCLRQ.2DA

  

ABRACEAD.2DA

  Modification to stats (Str,Dex etc.) at character creation

ABRACERQ.2DA

  Minimum and Maximum stat Per Race

ALIGNMNT.2DA

  Alignment per class

ANISUM0*.2DA (where * is the number)

  what types of monsters are summoned using a Monster Summoning spell. Same thing for ANISUM** first column covers the number in list and the second one the CRE file number.

AREALINK.2DA

  Contains the number of links and the resref of the link for each area file.

BACKSTAB.2DA

  assigns backstab modifier for classes (thief)

BNTYCHNC.2DA

  Bounty Chance? (The chance a party have of being attacked by Bounty Hunters)

CACHVALD.2DA

  the proper size of area BIFF files used for caching purposes probably not used in other games than BG1.........I don't know

CHPTXT*.2DA (where * is a number)

  description how to show chapter introduction text.
CHARSND.2DA The sounds for players in BG1 . The first column is filled by numbers meaning the slots from SOUNDOFF.IDS (so the actions of PC/NPC meaning) the next columns are filled by the Strrefs and labeled by name of soundset for PC

CHRMODST.2DA

  charisma modification

CLASWEAP.2DA

  Weapon restriction for classes

CSOUND.2DA

  identification of NPC sounds (in the soundsets, sounds are identified by a letter (male01a, male01b, etc...), but NPC sounds are identified by a number (kivan1, kivan2, etc...). This file convert the numbers to letters.)

DEATH.2DA

  if a NPC will react to the death of an other or not (BG1 only)

DEXMOD.2DA

   modification to Dexterity as it increases/decreases

DONARUMR.2DA

  Probability of rumor appearance when drinking alcohol in Inns ?

DRMTXT*.2DA (where * is the number)

  (* is a number)= text of the dreams (BG1 only)

EFFTEXT.2DA

  States of a character (Poisonned, Death, Petrification, etc...)

FATIGMOD.2DA

  luck modifier if a character is fatigued (penalty to attack/damage roll)

HAPPY.2DA

  reputation effect on the happiness of a character

HPCONBON.2DA

  Constitution modification

HP***.2DA (where *** is NPC class identifier)

  Hp bonus and rolls per level up

INTERACT.2DA

  Not used in other games than BG1, used to attach insult/compliment speeches between NPCs in BG1

INTERDIA.2DA

  dialogue file for romances and other inter-characters chat (usually B*****.dlg)

INTMOD.2DA

  Intelligence modification

INTOXCON.2DA

  recovery rate of an intoxicated character, based on his/her constitution

INTOXMOD.2DA

  penalty (luck) or bonus (moral) of an intoxicated character, based on how intoxicated he/she is.

ITEMEXCL.2DA

  Items that cannot be used at the same time, value 0 can be used value 1 cannot be used together.

LAYHANDS.2DA

  how much hit points are cured by a Paladin or a Monk using Lay on Hands at a determined level (2 at level 1, 4 at level 2, etc...)

LORE.2DA

  how many lore points a class gain per level

LOREBON.2DA

  bonus to lore points, determined by intelligence and wisdom

MASTAREA.2DA

  determines the TYPE name for area variables that work only in this area (type like GLOBAL, LOCALS, but with specified area with the name of it) 

MONSUM**.2DA (where ** is the number)

  what types of monsters are summoned using a Monster Summoning spell. Same thing for ANISUM**

MONTHS.2DA

  name and duration of each months

MOVIDESC.2DA

  title of each movie (when you want to view them with the View Movie option)

MXSPL***.2DA (where *** is NPC class identifier) 

 gaining spells tables per level

NPCLEVEL.2DA

  level of the NPC, based on yours (if you want to have Jaheira in your team and you are level  6, she will be level 6/7, etc...)

PDIALOG.2DA

  dialogue files and script files to use for an NPC

PORTRAIT.2DA

  replacement portrait (if my character has Khalid's portrait, which one will Khalid have in the game)

PROFS.2DA

  how many proficiencies * a character will have at his/her creation, and how often he/she will get an other one.

PROFSMAX.2DA

  the number of prof * a character can put in one weapon at his/her creation, and how many he/she can put after his/her creation

RAISDEAD.2DA

  how much it costs to raise a character of a determined level

REPMODST.2DA

  appears to control increases or decreases in the price of items depending on the party's reputation.  The top row of numbers indicates the party's reputation, and the number below shows the percentage of the cost from the item file that the player will have to pay.

REPSTART.2DA

  starting reputation (8 for chaotic evil, 12 for lawful good, etc...)

REPTXT.2DA

  "title" of a reputation level (17, 18, 19 and 20 are Heroic, 1 is Despised, etc...)

REPUTATI.2DA

  reputation modification based on actions (kill someone. etc...)

RMODCHR.2DA

  NPCs reaction modifier associated to the CRE charisma value

RMODREP.2DA

  NPCs reaction modifier associated to the CRE reputation value

RNDMAGIC.2DA

  a random magical item (for CREs for example)

RNDSCROL.2DA

  a random scroll (for CREs for example)

RNDTREAS.2DA

  random gems, coins or necklaces (used for CREs for example)

SAVE****.2DA (where **** is NPC class identifier)

  Save throws per class

SCRPDESC.2DA

  names and descriptions of the scripts

SKILLBRD.2DA

  Pick-pocket skill of a bard per level (probably possible adding other abilities, but wasn't checked)

SKILLDEX.2DA

  Dexterity bonus or penalties to thieving skills

SKILLRAC.2DA

  Racial bonus or penalties to thieving skills

SKILLRNG.2DA

  Hide in Shadows skill of a ranger per level (probably other could be added as well, not checked)

SNDCHANN.2DA

  determines volume of WAVC sounds used in various situations

SPELDESC.2DA

  description of the spells offered at temples

SPELLS.2DA

  ???????????????????????

SPRKLCLR.2DA

  probably the color of the sparks shown behind a projectile (used in PRO files)

STARTARE.2DA

  starting area with position of main character

STARTPOS.2DA

  starting position for games that uses more than one custom made character.

STATDESC.2DA

  description of the Special Effects Icons (charm, domination, poison, etc...)

STRMOD.2DA

  strength modification

STRMODEX.2DA

  strength modification features (strength=18/modificator)

STRTGOLD.2DA

  the amount of gold pieces that a class have at the beginning of the game (BG1 only)

THAC0.2DA

  THAC0 for each class at a determined level

TOOLTIP.2DA

  the text of the "special powers" extended headers (Haste and Invisibility of the Ring of Gaax, etc...)

WSPATCK.2DA

  Weapon speed attack. The left column goes to 5 for the proficiency points and the top is for your level. The numbers there are for extra attacks depending on level. 

WSPECIAL.2DA

  modification of weapon speed, damages inflicted and to hit bonus, based on weapon proficiencies.

XPCAP.2DA

 maximum amount of experience allowable for player  in game

XPLEVEL.2DA

  amount of xp needed to reach next level of the NPC. 

YEARS.2DA

  Sets starting game date  and  date format.
