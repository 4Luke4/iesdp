MUSIC.2DA

This file assigns the MUS file to the certain values. and is used by scripting and ARE files to determine what music should be played. Only available in IWD/HoW/ToT:L

The numbers used by ARE files song sections and scripting commands like PlaySong(number) are just the number of the identifiers on the list counting from 0 for the first identifier on the list. 

The first column contains of descriptions /identifiers (I have no idea if they can be used instead of values in scripts).

And second one contains of MUS files references that describes what ACM sound files are used and how to play song.
