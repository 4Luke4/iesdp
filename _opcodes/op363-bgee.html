--- n: 363 name: "Movement rate check" param1: "Movement Rate" param2: "Type" special: "Modal State" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Trying to move will break the current modal action, and breaking the modal action will remove this effect and any other effects applied by the same spell.
Known values for 'Type' are:
0 Cumulative Modifier -> Movement = Movement + 'Statistic Modifier' value
1 Flat Value Modifier -> Movement = 'Statistic Modifier' value
2 Percentage Modifier -> Movement = (Movement * 'Statistic Modifier' value) / 100

'Modal State' defines the modal state to check (see modal.ids).
