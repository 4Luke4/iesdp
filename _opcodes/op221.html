--- n: 221 name: "Removal: Remove Secondary Type" param1: "Maximum Level" param2: "Secondary Type" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Removes effects of the specified Secondary Type of up to the level specified by the Maximum Level field from the creature.

V1 effects inherit their Secondary Type from the Header(SPL) / Extended Header(ITM), V2 effects specify it manually.


Known values for Secondary Type are:
  • 0 ⟶ NONE
  • 1 ⟶ SPELLPROTECTIONS
  • 2 ⟶ SPECIFICPROTECTIONS
  • 3 ⟶ ILLUSIONARYPROTECTIONS
  • 4 ⟶ MAGICATTACK
  • 5 ⟶ DIVINATIONATTACK
  • 6 ⟶ CONJURATION
  • 7 ⟶ COMBATPROTECTIONS
  • 8 ⟶ CONTINGENCY
  • 9 ⟶ BATTLEGROUND
  • 10 ⟶ OFFENSIVEDAMAGE
  • 11 ⟶ DISABLING
  • 12 ⟶ COMBINATION
  • 13 ⟶ NON-COMBAT
{% capture note %} Will not remove effects that use Timing Mode 9. {% endcapture %} {% include important.html %} {% capture note %} If any effects are removed, the String Reference corresponding to the Secondary Type from MSECTYPE.2DA is returned to the combat log. {% endcapture %} {% include info.html %}
