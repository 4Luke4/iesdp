--- n: 16 name: "State: Haste" param1: "Irrelevant" param2: "Type" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Applies the Haste effect, in the style specified by the 'Type' value.
Known values for 'Type' are:
0 Normal Haste
1 Improved Haste
2 Weak Haste – Speed haste without attack bonuses

All of them set STATE_HASTED. Improved/Weak function by setting an additional STAT value, which modifies the behavior of STATE_HASTED:
  • STAT 155 – IMPROVEDHASTE.
  • STAT 185 – ANIMATION_ONLY_HASTE.

This only works when applying normal/improved haste (there's no difference, since the STAT for improved haste doesn't get set this way), weak haste does not do anything with timing mode 1. {% endcapture %} {% include warning.html %}
