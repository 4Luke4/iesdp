--- n: 99 name: "Spell Effect: Duration Modifier" param1: "Duration Modifier" param2: "Type" bg1: 1 bg2: 1 bgee: 0 iwd1: 1 iwd2: 1 pst: 1 --- Alters spell durations of the targeted creature(s), to the percentage specified by the 'Duration Modifier' field, affecting the spell type specified by the 'Type' field. Timing mode 0 is the only timing mode reliably affected. Timings handled through PRO files are ignored.
eg.
'Duration Modifier' = 0 -> Spells have no duration
'Duration Modifier' = 100 -> Spells have normal duration
'Duration Modifier' = 200 -> Spells have double duration
Known values for 'Type' are:
0 Mage
1 Priest {% capture note %} Not seen to work in BG1. {% endcapture %} {% include note.html %}
