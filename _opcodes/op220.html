--- n: 220 name: "Removal: Remove School" param1: "Maximum Level" param2: "School Type" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Removes effects of the specified School Type of up to the level specified by the Maximum Level field from the creature.

V1 effects inherit their School Type from the Header(SPL) / Extended Header(ITM), V2 effects specify it manually.


Known values for School Type are:
  • 0 ⟶ NONE
  • 1 ⟶ ABJURATION
  • 2 ⟶ CONJURATION
  • 3 ⟶ DIVINATION
  • 4 ⟶ ENCHANTMENT
  • 5 ⟶ ILLUSION
  • 6 ⟶ EVOCATION
  • 7 ⟶ NECROMANCY
  • 8 ⟶ ALTERATION
  • 9 ⟶ GENERALIST
  • 10 ⟶ WILDMAGE
{% capture note %} Will not remove effects that use Timing Mode 9. {% endcapture %} {% include important.html %} {% capture note %} If any effects are removed, the String Reference corresponding to the School Type from MSCHOOL.2DA is returned to the combat log. {% endcapture %} {% include info.html %}
