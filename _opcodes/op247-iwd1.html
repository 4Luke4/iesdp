--- n: 247 name: "Beltyn's Burning Blood" param1: "Strikes Number" param2: "Irrelevant" bg1: 0 bg2: 0 bgee: 0 iwd1: 1 iwd2: 0 pst: 0 --- Applies the beltyns burning blood effect to the targeted creature(s).
The affected creature(s) go bersek for the number of attack specified by the 'Strike Number' field, on a failed saving throw vs spells.
Note: In IWD (with no expansions) param1 acts as a bonus to damage, rather than a strike number.
Note: This effect can lead to a crash upon save game loading, if a creature is killed due to this effect.
