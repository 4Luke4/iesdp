--- n: 179 name: "Spell Effect: Damage vs. Creature Type Modifier" param1: "Reference Value" param2: "Reference IDS" special: "Statistic Modifier" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Alters the targeted creature(s) damage against the creature type specified by the 'IDS Entry' field in the specified 'IDS File'. The actual modifier value is placed in the Parameter 3 field in an external effect - dice values are ignored.
Known values for 'IDS File' are:
2 EA.ids
3 General.ids
4 Race.ids
5 Class.ids
6 Specific.ids
7 Gender.ids
8 Align.ids
9 Kit.ids (EE only) {% capture note %}
  • It will affect all attacks, not just the weapon it's on, which would be incorrect when dual-wielding.
  • For two-handed/ranged weapons, it can be ideal if a base weapon damage bonus is desired.
  • Damage is multiplied by critical hits (and maybe backstabs).
  • Multiple opcode #179 effects only allow additional targeting, they do not stack their damage bonus.
