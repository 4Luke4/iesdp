--- n: 336 name: "Graphics: Display Eyes Overlay" param1: "Sequence" param2: "Eye Group" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Displays a twin VVC or BAM linked to the target.
Known values for 'Sequence' are:
  • 0 => None
  • 1 => A
  • 2 => B
  • 3 => C
  • 4 => D
  • 5 => E
  • 6 => F
  • 7 => G

'Eye Group' is related to opcode #335.
The two animations that are played must be named in the following way:
  1. 'Resource'1'Sequence'.'extension'
  2. 'Resource'2'Sequence'.'extension'
where 'Resource' is what you need to put in the resource field of this effect.
For instance, in IWD:EE we have: 
	#7eyes1a.VVC
	#7eyes1b.VVC
	#7eyes1c.VVC
	#7eyes1d.VVC
	#7eyes1e.VVC
	#7eyes1f.VVC
	#7eyes1g.VVC
	#7eyes2a.VVC
	#7eyes2b.VVC
	#7eyes2c.VVC
	#7eyes2d.VVC
	#7eyes2e.VVC
	#7eyes2f.VVC
	#7eyes2g.VVC
Effect terminates when 'Sequence' = 0.
