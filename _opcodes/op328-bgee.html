--- n: 328 name: "State: Set State" param1: "Irrelevant" param2: "Type" special: "Mode" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Sets the state of the targetted creature(s) to the value specified by the 'Type' field.
These effects could be detected by the gamescript trigger ExtendedStateCheck. (see extstate.ids).
Known values for 'Type' are:
0 Chaotic Command
1 Miscast Magic
2 Pain
3 Greater Malison
4 Blood Rage
5 Cats Grace
6 Mold Touch
7 Shroud of Flame
If 'Mode' was set to 1, then this effect behaves like IWD2 Set State effect. (see splstate.ids)
