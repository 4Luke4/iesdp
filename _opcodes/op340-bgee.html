--- n: 340 name: "Spell Effect: Change Backstab Effect" param1: "Irrelevant" param2: "Irrelevant" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Sets the Back Stab effect of the targetted creature to the spell specified by the resource key. Both parameters should be 0 (zero).
In particular:
  • If the game option '3E Sneak Attacks' is not enabled, the spell is cast at the character's innate level.
  • If the game option '3E Sneak Attacks' is enabled, the spell is cast at level defined in CRIPPSTR.2DA + 1.
    Default spell (without this effect) is "BACKSTAB.SPL".
