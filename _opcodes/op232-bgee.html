--- n: 232 name: "Spell Effect: Cast Spell on Condition" param1: "Target" param2: "Condition" special: "Extra" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Casts the spell specified by the resource field on the target specified by the 'Target' field, when the condition from the 'Condition' field is true. The conditions are akin to various triggers.
Known values for 'Target' are:
0 Effect Target
1 LastHitter
2 NearestEnemyOf
3 Nearest

Known values for 'Condition' are:
0 HitBy()
1 See(NearestEnemyOf())
2 HPPercentLT(Myself 50)
3 HPPercentLT(Myself 25)
4 HPPercentLT(Myself 10)
5 StateCheck(Myself,STATE_HELPLESS) i.e. unconscious
6 StateCheck(Myself,STATE_POISONED) i.e. poisoned
7 AttackedBy('Target')
8 PersonalSpaceDistance('Target',4)
9 PersonalSpaceDistance('Target',10)
10 Unknown - every round
11 TookDamage()
12 Kill made
13 Time of Day is 'Extra'
14 Target in 'Extra' range
15 Target's state is 'Extra'
16 Target dies
17 Target died
18 Target turned by
19 Target HP < 'Extra'
20 Target HP % < 'Extra'

The 'Extra' field is used for the following 'Target' values:
13: Use value from TIMEODAY.IDS
14: Range in feet
15: Use value from STATE.IDS
18: Any numeric value
20: Any numeric value

For a 'Condition' of 13, the 'Target' parameter must be 0.

Parameter 3 (EFF) activates the portrait icon and immediately triggers the HitBy() trigger
