CHPTXT*.2DA (where * is the number)

This file defines how to show chapter introduction text

First name (in our example ch05aund ) determines BMP reference for chapter screen..

The first column is filled by race names.

Later there are two columns that could be filled by strrefs from Dialog.tlk.
