HP****.2DA (where stars are identifiers of classes)

This file defines HP rolls and bonus per CRE level.

There are following files of this type :

  • HPWIZ.2DA (for wizards, mages , sorcerers)
  • HPPRS.2DA (for priests, druids)
  • HPMONK.2DA (for monk)
  • HPWAR.2DA (for warriors, fighters, paladins, rangers)
  • HPROG.2DA (for rogues - thieves, bards)

Where :

First colunm is the PC's/NPC's level.

Columns 2-4 are obvious from their names (number of dice sides, number of rolls and their modifier)
