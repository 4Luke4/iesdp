SKILLRAC.2DA

This file defines racial bonus or penalties to thieving skills

The first row (titles of columns ) is filled by the names of features affecting of PC (like PICK_POCKETS for example ).

The first column is filled by races identifiers.

The rest columns are filled by modifiers for given  race - pretty obvious from names I suppose.
