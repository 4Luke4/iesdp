BACKSTAB.2DA

This file assigns backstab modifiers to the according thief levels (I have no idea if it will change anything if somebody will add other class and give it backstab modifiers.)

On example of  BG1 we have :

first row contains of levels of the PC/NPC

the second one , entitled thief contains of the values of backstab modifier. It is unknown however if adding name of other class would give it backstab possibility.
