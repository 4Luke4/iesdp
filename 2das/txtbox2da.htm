SCRSARA.2DA (And also other 2DA files that are linked here)

This file defines how to show text box

First name (in our example CHPTSARA ) determines BMP reference for text screen..

The first column is filled only by DEFAULT identifier..

Later there are two columns that could be filled by strrefs from Dialog.tlk.

This file is from BG2 SoA /ToB :

2DA V1.0
CHPTSARA
                        0           1
SWITCH                  DEFAULT     DEFAULT
DEFAULT                 16208       71590

There is also one more type of such scenes :

SCENE0*.2DA (where * is a number)

DRMTXT*.2DA (used only in BG1 for dreams, * is a number)

KPLOSE.2DA (for BG2 only)

KPWIN.2DA (for BG2 only)

KPREVOLT.2DA (for BG2 only)

Is just example , does the same thing like previous example , but has a bit different options - probably they can be also implemented in previous files , as their structure is very similar.

the first name below file signature is name of graphics MOS file to be shown in text box.

in column marked by 1 we have title reputation , that indicates reputation role in modification of this feature.

The first column consists of reputation identifiers an in row we have the place of strrefs from Dialog.tlk that shows appropriate text according to reputation.

-1 value as always disable given feature
