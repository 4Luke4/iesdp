Lists of 2DA files according to the given Infinity Engine game

 

Baldur's Gate 2 SoA

 

2DA file name short description

ABCLASRQ.2DA

  minimum stats (Str,Dex etc.) per class/kit

ABCLSMOD.2DA

  modification to stats per class

ABDCDSRQ.2DA

  Minimum stat required to dual-class with this class

ABDCSCRQ.2DA

  Minimum stat required to be able to dual-class

ABRACEAD.2DA

  Modification to stats (Str,Dex etc.) at character creation

ABRACERQ.2DA

  Minimum and Maximum stat Per Race

ABSTART.2DA

  Special abilities at the beginning of BG2 SoA for new characters (not imported from BG1)

A***.2DA (where stars are creature abbreviation)

  Sounds linked to the creature avatar (linked to ANISND.IDS)

ALIGNMNT.2DA

  Alignment per class
ANIMSND.2DA  There is a wild surge effect that gives hiccup to the caster. Whenever he/she is supposed to make a sound, there is a little hiccup noise. This file could be the soundset for hiccup (like Amoo is the soundset for mooses).

ANISUM0*.2DA (where * is the number)

  what types of monsters are summoned using a Monster Summoning spell. Same thing for ANISUM** first column covers the number in list and the second one the CRE file number.

AREALINK.2DA

   Contains the number of links and the resref of the link for each area file.
AREALINS.2DA  

BACKSTAB.2DA

  assigns backstab modifier for classes (thief)

BNTYCHNC.2DA

  Bounty Chance? (The chance a party have of being attacked by Bounty Hunters)

CACHVALD.2DA

  the proper size of area BIFF files used for caching purposes probably not used in other games than BG1.........I dunno
CHARSND.2DA  sounds of characters (players) references in dialog.tlk, probably were used only in BG1 (if at all)

CHPTXT*.2DA (where * is a number)

  description how to show chapter introduction text.

CHRMODST.2DA

   charisma modification
CLAB****.2DA (where **** are specific kit identifier)  List of the special abilities of each kit.

CLASWEAP.2DA

  Weapon restriction for classes
CLEARAIR.2DA  type of stationary effect that are cleared by the Zone of Sweet Air spell
CONTCOND.2DA  What condition triggers the contingency
CONTINGX.2DA  What spells can't be placed in contingency
CONTTARG.2DA  Contingency target

CSOUND.2DA

   identification of NPC sounds (in the soundsets, sounds are identified by a letter (male01a, male01b, etc...), but NPC sounds are identified by a number (kivan1, kivan2, etc...). This file convert the numbers to letters.)

DEATH.2DA

  if a NPC will react to the death of an other or not (BG1 only)

DEXMOD.2DA

   modification to Dexterity as it increases/decreases

DONARUMR.2DA

  Probability of rumor appearance when drinking alcohol in Inns ?
DUALCLASS.2DA  Dual-classing possibilities On the crosses of the columns and rows names 0s indicates what classes cannot be dualclassed and 1s the ones that can be.

EFFTEXT.2DA

  States of a character (Poisonned, Death, Petrification, etc...)
ENDGAME.2DA  Has something to do with the end movie (297 in the Dialog.tlk is Finale and 298 is ***Placeholder for end movie***)

FATIGMOD.2DA

  luck modifier if a character is fatigued (penalty to attack/damage roll)

HAPPY.2DA

  reputation effect on the happiness of a character
HATERACE.2DA  Attaches text strings (strrefs from dialog.tlk) describing racial enemy of ranger. 

HPCONBON.2DA

  Constitution modification
HP****.2DA (   where **** is the NPC class identifier)  Hp bonus and rolls per level up
IMPORT0*.2DA (where * is the number)  List of items that can be imported from BGI to BG2, and placed in three different places in Irenicus' dungeon
INTERACT.2DA  Not used in other games than BG1, used to attach insult/compliment speeches between NPCs in BG1

INTERDIA.2DA

  dialogue file for romances and other inter-characters chat (usually B*****.dlg)

INTMOD.2DA

  Intelligence modification

INTOXCON.2DA

  recovery rate of an intoxicated character, based on his/her constitution

INTOXMOD.2DA

   penalty (luck) or bonus (moral) of an intoxicated character, based on how intoxicated he/she is.
ITEMANIM.2DA  
ITEMDIAL.2DA  Item dialogue (Lilarcor the talking two-handed sword)

ITEMEXCL.2DA

  Items that cannot be used at the same time, value 0 can be used value 1 cannot be used together.
K_*_*.2DA (where * are identifiers for classes and races)  which kit a determined race can use
KITLIST.2DA  Different things related to kits (item restrictions, descriptions, etc...)
KITTABLE.2DA  List of the different kit files (K_*_*) by class
KPLOSE.2DA  Text displayed when the Roenals take control of the keep (fighter's stronghold)
KPREVOLT.2DA  Text displayed when the villagers take control of
the keep (fighter's stronghold)
KPWIN.2DA  Text displayed when you defeat the Roenals (fighter's stronghold)

LAYHANDS.2DA

  how much hit points are cured by a Paladin or a Monk using Lay on Hands at a determined level (2 at level 1, 4 at level 2, etc...)
LOADHINT.2DA  Load hints (the hints displayed when loading or saving a game)

LORE.2DA

  how many lore points a class gain per level

LOREBON.2DA

  bonus to lore points, determined by intelligence and wisdom
LVLMODWM.2DA  
MASTAREA.2DA  determines the TYPE name for area variables that work only in this area (type like GLOBAL, LOCALS, but with specified area with the name of it) 
M*****.2DA (where ***** is the monster identifier)  Sounds linked to the creature avatar (linked to ANISND.IDS)

MONSUM**.2DA (where ** is the number)

  what types of monsters are summoned using a Monster Summoning spell. Same thing for ANISUM** 

MONTHS.2DA

  name and duration of each months

MOVIDESC.2DA

  title of each movie (when you want to view them with the View Movie option)

MSCHOOL.2DA

  the text that appears after dispelling spell from the given mage school

MSECTYPE.2DA

  the text that appears after removing some defined spell protections (like contingency, battle protections) by other spell

MXSPL***.2DA (where *** is the NPC class identifier)

  gaining spells tables per level

NPCLEVEL.2DA

  level of the NPC, based on yours (if you want to have Jaheira in your team and you are level  6, she will be level 6/7, etc...)

PDIALOG.2DA

  dialogue files and script files to use for an NPC

PORTRAIT.2DA

  replacement portrait (if my character has Khalid's portrait, which one will Khalid have in the game)

PROFS.2DA

  how many proficiencies * a character will have at his/her creation, and how often he/she will get an other one.

PROFSMAX.2DA

  the number of prof * a character can put in one weapon at his/her creation, and how many he/she can put after his/her creation
RACEHATE.2DA  

RAISDEAD.2DA

  how much it costs to raise a character of a determined level
RANDCOLR.2DA  random colors of some specific creatures avatars

REPMODST.2DA

  appears to control increases or decreases in the price of items depending on the party's reputation.  The top row of numbers indicates the party's reputation, and the number below shows the percentage of the cost from the item file that the player will have to pay.

REPSTART.2DA

  starting reputation (8 for chaotic evil, 12 for lawful good, etc...)

REPTXT.2DA

  "title" of a reputation level (17, 18, 19 and 20 are Heroic, 1 is Despised, etc...)

REPUTATI.2DA

  reputation modification based on actions (kill someone. etc...)

RMODCHR.2DA

  NPCs reaction modifier associated to the CRE charisma value

RMODREP.2DA

  NPCs reaction modifier associated to the CRE reputation value
RNDEQUIP.2DA  Random equipment (same as RNDWEP, usage not determined yet)

RNDMAGIC.2DA

  a random magical item (for CREs for example)

RNDSCROL.2DA

  a random scroll (for CREs for example)

RNDTREAS.2DA

  random gems, coins or necklaces (used for CREs for example)
RNDWEP.2DA  Random weapon, armor or shield

SAVE****.2DA (where **** is class identifier)

  Save throws per class
SCENE0*.2DA (where * is the number)  Next text box definitions depending sometimes on reputation

SCRPDESC.2DA

  names and descriptions of the scripts
SCRTXT0*.2DA (where * is the number)  contain of the text box information for chapters and other quests. 

SKILLBRD.2DA

  Pick-pocket skill of a bard per level (probably possible adding other abilities, but wasn't checked)

SKILLDEX.2DA

  Dexterity bonus or penalties to thieving skills

SKILLRAC.2DA

  Racial bonus or penalties to thieving skills

SKILLRNG.2DA

  Hide in Shadows skill of a ranger per level (probably other could be added as well, not checked)

SNDCHANN.2DA

  determines volume of WAVC sounds used in various situations
SNDENVRN.2DA  assigns to specified identifiers the sounds playing parameters (not only volume)
SNDRESRF.2DA  assigns identifiers from SNDENVRN.2DA to areas in aim to modify sounds playing on the specified area
SONGLIST.2DA  list of songs in game (acm files) with attached reference  numbers (BG2/ BG2 ToB only)
SPAWNGRP.2DA  

SPELDESC.2DA

  description of the spells offered at temples

SPELLS.2DA

  
SPIRSUMM.2DA  Works as ANISUM**, but for the Spirit animal summoning of the Totemic Druid.
SPLASHSC.2DA  shows advertisement screens.
SPLAUTOP.2DA  Random list of spells used in the Auto-pick function when creating a mage or sorcerer.
SPLSRCKN.2DA  Spells table progress per level  for sorcerer class
SPRKLCLR.2DA  probably the color of the sparks shown behind a projectile (used in PRO files)
SPWI***.2DA (where *** is the number) (or any other SPL based filename)  These 2DA files base on effect D6h (select spell) and they are simply lists of these spells for the another items or spells ( first column covers some identification names, second one covers the resrefs to the SPL files and third one labeled TYPE is unknown )

STARTARE.2DA

  starting area with position of main character

STARTPOS.2DA

  starting position for games that uses more than one custom made character.

STATDESC.2DA

   description of the Special Effects Icons (charm, domination, poison, etc...)

STRMOD.2DA

  strength modification

STRMODEX.2DA

  strength modification features (strength=18/modifier)

STRTGOLD.2DA

  the amount of gold pieces that a class have at the beginning of the game (BG1 only)

THAC0.2DA

  THAC0 for each class at a determined level

TOOLTIP.2DA

  the text of the "special powers" extended headers (Haste and Invisibility of the Ring of Gaax, etc...)
TRANDEF.2DA  
WEAPPROF.2DA   allowed number of stars in weapon proficiencies, their names and description strrefs.

WSPATCK.2DA

  Weapon speed attack. The left column goes to 5 for the proficiency points and the top is for your level. The numbers there are for extra attacks depending on level. 

WSPECIAL.2DA

  modification of weapon speed, damages inflicted and to hit bonus, based on weapon proficiencies.
XPBONUS.2DA xp given to NPC/PC for performing special action like disarming traps

XPCAP.2DA

 maximum amount of experience allowable for player  in game

XPLEVEL.2DA

 amount of xp needed to reach next level of the NPC. 

YEARS.2DA

 Sets starting game date  and  date format.
