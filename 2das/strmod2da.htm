STRMOD.2DA

This file defines strength influence on characters abilities like allowable weight that could be taken by character or to hit bonus

The first row (titles of columns ) is filled by the names of features affecting of PC (like TO_HIT = to hit bonus ).

The first column is filled by value of strength of character.

The rest columns are filled by values for given  strength  - pretty obvious from names I suppose.
