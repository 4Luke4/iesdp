IE acronyms and definitions

 

 

Death variables    are the names used in scripts to control creature's actions. For example, Minsc has about 3 different creature files, all with the same death variable, "Minsc," in the creature file. So, any of the Minsc creature files would respond to "Minsc" as an object in a script. The death variables are stored in the cre files
