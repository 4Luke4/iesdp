How to create new Racial Enemies

By Daxziz (with some minor corrections of me)

It is all very simple. First will you have to find the file called HATERACE.2da.

As you can see off its name, does it contain the RACEs which is HATEable... modern said Racial Enemies!

The next file you will have to find is RACE.IDS, this is the one which I think contains all the races, and might also help in the future, to possible creation of brand new races... but we will see.

1: Open the HATERACE.2da in a text editor as Note- or Wordpad.

2:  Scroll down to the end of the file, there should be this line:

VAMPIRE                 54779       125     54780

3: Add the name of your new enemy just under VAMPIRE.

We take elf as an example!

VAMPIRE                 54779       125     54780

ELF

4: Open the Infinity Engine Dialog.tlk Editor.

Push, and select, the Write (ratio) button.

Hit the Write button in the side.

5: Write: Elf

Take note of the resource number which appears.

6: Just under the vampires STRREF number, do you write the new one you got from the Dialog.tlk Editor.

It should now look something like this

      - Warning the number you have gotten might be different -

VAMPIRE                 54779       125     54780

ELF                     62276

7: Now open the RACE.IDS (RACE.LUA when using old biffstripper) file, also in a text editor, and find the new enemy, in our case is it number 2. Add it under the 125 of the Vampire. ( Note if you scroll down the RACE.IDS file, will you see, that VAMPIRE have the number 125 in front of it.

Now should it all look something like this:

VAMPIRE                 54779       125     54780

ELF                     62276         2    

8: Go back to the Infinity Dialog.tlk Editor, and again push, and select, the Write (ratio) button.

Write a little info about your new racial enemy... perhaps the info which you can find in a AD&D book... just an idea.

Hit the Write button in the side.

Remember the number you are given.

9: Paste the number beneath the last number from the Vampire (54780) in the HATERACE.2da file.

It should look something like this:

VAMPIRE                 54779       125     54780

ELF                     62276         2     62277

10: Save the changes you have made in the file, and put it in your Override folder, in the Baldur's Gate 2 directory.
