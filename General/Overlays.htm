How to manage overlays in WED files ?

Overlays Tiles lists

These tiles have usually dimensions 1x1 and they contain several "frames" when they are animations. Also one good thing worth noted is that these tiles are used only in BG1 and BG2. I haven't found any file in IWD and PST that use them (though I haven't checked HoW and ToTL) . However of course using them in these games is no problem , but they should be added then either to BIF file/chittin.key or to Override folder of appropriate IE game

Their names are entered in WED files Overlays section appropriate offset.   

Their handling and using on the main overlay are described respectively in WED and TIS file formats.

As for one thing - all tile sets named per *R.tis seem to be tile sets made for raining case. Though I have no idea if it works in game in such a way at the moment (bubbles that may be observed in these "R" tis files suggest this as well as using "R"- like rain identifier in name)

In OVERLAYS section :

0x0004 8 (resref) Name of tileset (resource type 0x3eb)

So we have the following possibilities by default :
