Music and Audio

Q: Where to place custom songs?

A : ACMs are put only in IEgame\Music\FolderName folder

Where FolderName is defined by MUS file that is located in IEGame\MUSIC folder.

For game speeches and other FX sounds WAVCs or WAVs are used , not ACMs. 


A : If you think about sound files then you use WAV and WAVC files for speeches, spells effect sounds, other fx (put this in OVERRIDE folder) or ordinary WAVs in SOUND folder with proper naming convention for making PCs soundsets (there is readme file attached to game what names)

or finally area/other music - in MUSIC folder there are used ACM files with their playlists (MUS files)

Description of all these things you should find either in game readme (players soundsets) or in appropriate formats description (speeches/fx WAVC/WAV sounds and ACM music)

Q : How can I use animation of monk's kicking / punching in my kit ?

A : To use the monk avatar's kung-fu fighting animation, you must make a two-handed fist. The martial art kicking and punching sequences match with the two-handed sequences of the other avatars. That the monk's fist is two-handed explain why a monk can't use a shield or a second weapon, and the fact that the two-handed weapon animation is used for the kung-fu explain why a monk can't use two-handed weapons, and that a fighter with a staff (or a two-handed sword) and a monk avatar will kick and punch. So that means that only the specific moments when the fists are given is hardcoded

IE scripting

Q :  What is the difference between the .bs and .baf files.

A : Baf is the uncompiled rough file, bs is the compiled format. You can rename bs extensions bcs to use in the game

Top

Q : How to learn scripting ?

A : An excellent way to learn scripting (the way I learned the most) is to use Infinity Explorer to look at the scripts the games itself is using and see what is doing what. Most of the scripts are labeled by the name of the creature that uses them or the area. So for example Korgan's script is called Korgan.Bcs. I still have a lot to learn about scripting because the game itself only takes it so far. And alot of the commands (Triggers, actions,etc) are rarely used so its hard to see an example of them in use to learn how to do it.

I recommend going to bg scripts and using the search button to check the archives for topics pertaining to what you want to learn (For example Enter 'Specifics' to find all messages pertaining to specifics).

Also join one or both of the scripting groups listed in Links section and you can set your emails to receive any topics discussed, so you can simply listen to people talk about scripting.

Q : Without changing the graphics, how do you get a spell to NOT affect your party? (ie. bad guys only)

A : To have it not affect your party, you must a have a trigger protection from spell (or effect), and the spell resource name (or effect) you want to protect against, and have the target set to party.

Q : Using IE I have found numerous files in the spell list that have no name or description only the file # such as SPPR609D. Does any one have a clue what these files would be or a way of finding out?

A : Some of that stuff ends up being like spell versions of spells made for specific CRE files or stuff that was started but never finished or sometimes it is the other parts of a spell itself that is used from the original spell. 

Certain files... like for Blade Barrier have 2 files. One is for the Blade Barrier effect and the other is for the damage it does. So, usually if you have something like SPPR598 (made it up) and SPPR598a , then the first will be for the Blade Barrier effect and the second for the damage.

Many other spell files are also used for applying effects to kits.

Q : How do I target spells at specific creature types? I have made a Banishment-type spell that I have put in my Demon Summoning ring, but it will work against anything.

A : In general there are only three ways to target specific things, alignment, class, and race.  Anything in the corresponding *.ids files is targetable. So in this case, you can target them by class CLASS.IDS - 160 (tanari) or by race  RACE.IDS - 121 (demonic).
I cant really recommend a specific struct to use, but I suspect you will have to make a custom eff file to use to get the effect you are looking for, since the information above is pretty much only of use if you are looking  to modify an existing effect (like an arrow of slaying ogre mages to slaying demons, etc).

Q : What are vertices ? I know it is associated with areas but what is it? 

A : Vertices are used to make the shapes of triggers, containers, etc...

Basically it draws an outline like on a graph. Draw line from point [1.1] to [1.2] to [2.2] to [2.1] and you get a square. 

In general, a vertex (pl. vertices) is a term used in geometry to mean corner. Eg. a square has 4 vertices. Simple innit? 
It is also used in terminology for graphs where each vertex is a point of data on the graph.

Q : I've done an area and everything works fine. Except for one thing, I cannot travel from the area. It is accessible from other areas. I saw in another topic that to get the cursor to change to the travel cursor when you go to the edge of the map, you had to have blue lines around the search map. I have seen those blue lines on maps extracted from the game. But in all maps I start to do from scratch those blue lines never show up. And I can for my life not figure out how to get them there. I'm using IETME and I have read the tutorial that comes with it. What am I missing?

A : Open the area's search map, xxxxxSR.bmp, with a paint program, (e.g. pbrush, psp,...). Draw a 2 pixel wide border around the edge of the bitmap using an RGB value of  0,255,255.  This should allow you to travel out of the area on the world map.
