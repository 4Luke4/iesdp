How to make your own worldmap icon ?

 

Disclaimer : this tutorial should be regarded as developer notes more than as "tutorial" - it is just remainder of known stuff - Thie may require from reader some wisdom about hex editing, IESDP file formats descriptions interpretation ability, so don't be surprised if you don't understand something.

First of all you should start by finding out what is the name of the worldmap file that you want to modify : worldmap.wmp or wmap25.wmp

When you have it you either have possibility to check the Worldmap Icons resource in the Infinity Explorer or Near Infinity or in the Hex editor as it is described in WMP file format.

As for the the details : if you use hex editor check the name in offset 0x0030 of Worldmap section - this is name of needed BAM resource with icons.

It may be found in GUIBAM.BIF (or 25GUIBAM.BIF in case of ToB)

Then we use BAMWorkshop to modify worldmap resource. We either add new sequence that wil be our new Icon place or modify existing one.

After putting all needed icons in Sequence just remember sequence number and later, after putting new area on the map using ICM you have to hex into the number of sequence in hexadecimal system in the offset  0x0034 of area entry of your chosen area ( Just look for your area name within WMP file that is start of area section of it.

And that's all , remember to put your BAM either in override or in some bif file so game will know where to look for the BAM icon resources.
