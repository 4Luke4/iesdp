TOT file description

 

FileExtension : TOT (Talk Table Override Text)
Content : Description of TOT
History : File done by Manveru 06.09.2001 

A way to edit bios in IWD during game (Character not exported) . Actually this type of files is similar to TLK files and may be also used to override text from Dialog.tlk 

This file consists of String entries that are referenced from TOH files by the offset number. 

In BG2 it has other purpose : It is used to store all custom players comments inside game - it means custom waypoints on the area map, custom notes in journal as well as the biography that is stored in these files during game (character not exported). 

These files are parts of SAV files in Savegame folders. Connected with TOH files.

However these files have probably fixed length of string sections : I have only noticed sections that are 20Ch bytes long and :

Offset Size (data type) Description
0x0000 4 (dword) unknown , filled by FFFF FFFF or 0000 0000
0x0004 4 (dword) unknown, filled by FFFF FFFF or 0000 0000
0x0008 512 (char array) string data
0x0208 4 (dword) unknown, filled by FFFF FFFF

Where offset 0x0000 of given section is referenced from TOH file The TOT files contain of only these sections.
