TOH file description

 

File Extension : TOH (Talk Table Override Header)
Content : Description of TOH
History : File made by Manveru 06.09.2001 

A way to edit bios in IWD during game (Character not exported). Actually this type of files is similar to TLK files and may be also used to override text from Dialog.tlk 

This file is used as , say, "map" for the TOT file consisting of string references. It defins probably also more sophisticated features of string entries , like sound references (but it is not confirmed so far). At least there is possibility to override existing dialog.tlk strrefs using this file (for example custom biographies are set this way)

In BG2 it has other purpose : It is used to store all custom players comments inside game - it means custom waypoints on the area map, custom notes in journal as well as the biography that is stored in these files during game (character not exported). 

These files are parts of SAV files in Savegame folders. TOH is connected with TOT files.

These files could be obtained in uncompressed form by leaving game through other means (not elegant - switching off power when game is running). So you may find uncompressed SAV file in Temp directory in your game folder. 

Header :

This section is 14h bytes long

Offset Size (data type) Description
0x0000 4 (char array) Signature ('TLK ')
0x0004 4 (char array) nothing, probably place for version
0x0008 4 (dword) unknown
0x000c 4 (dword) Number of Strref entries in TOT file.
0x0010 4 (dword) unknown

Strref entries :

These sections have 1Ch bytes.

Offset Size (data type) Description
0x0000 4 (dword) string reference number. In TOH files it has values greater than 3000000 (2DC6C0h) starting from this number. Sometimes there are used smaller value to override existing Dialog.tlk entry
0x0004 4 (dword) unknown
0x0008 4 (dword) unknown
0x000c 4 (dword) unknown.
0x0010 8 (resref?) WAV/WAVC reference ?
0x0018 4 (dword) Offset to string data inside TOT file
