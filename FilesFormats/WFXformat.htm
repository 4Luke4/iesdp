WFX file format

FileExtension : NONE
Content : The description of WFX
History : Preliminary description by Manveru (12.07.2001)

These files have constant length 108h (264) bytes.  These files were found thanks to Gorin (for BG2)  :
     IMPPP0X. where x=1-7 in CRESound.bif.
     EFF_M21. in SFXSound.bif.

So as we can see there are only several examples of these files. Exact purpose is unknown, however they are located in Biffs that contain of huge amounts of Wavc files - so maybe they are responsible somehow for sounds ?

Offset Size (data type) Description
0x0000 4 (char array) Signature ('WFX ')
0x0004 4 (char array) Version ('V1.0')
0x0008 4 (dword) unknown, always 0
0x000c 4 (dword) unknown, observed values : 0 for EFF_M21 from SFXSound.bif and 4s for WFXs from CRESound.bif
0x0010 4 (dword) unknown, observed values : 0 for EFF_M21 from SFXSound.bif and 2s for WFXs from CRESound.bif
0x002c 244 (bytes) unknown, unused ? always 0.
