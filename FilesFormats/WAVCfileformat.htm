CONVERSION from WAV to WAVC

FileExtension : WAVC , WAV , WAC (Only files with WAV extension are recognized by game)
Content : Desription of WAVC (sound files) V.1.0
History : File done by Manveru Telcontar
History : Small addition of MaltanarTeamBG

Unlike ACM files, they can be put in the override folder, or BIF files. All WAVC files are 22050 Hz, 16 bits and mono.
Since the game only recognizes files with WAV extensions, they must be renamed to *.WAV before being used in the game.

Wavc is the sounds file format used in IE games. Actually it is an acm file with the additional header attached.
It has the following header :

0000x00 5741 4643 5631 2e30 xxxx xxxx yyyy yyyy
0000x10 1c00 0000 0100 1000 2256 e777 9728 0301


where :

Offset Size (Data Type)  Description
0000x00 8 (ASCIIZ) File identifier WAVCV1.0
0000x08 4 (dword) uncompressed size of file ( xxxx xxxx ) value
0000x0a 4 (dword) compressed size of file ( yyyy yyyy ) value
0000x10 12 (bytes) constant in all wavc files
0000x1c 4 (bytes) there is the header of normal acm file ( 9728 0301 )

In aim to make Wac  file we need the following utilities :
snd2acm.exe
acm2wav.exe
wavc.exe

I suppose that you could find these utilities on Potencius page or you could try on Russian Fallout customization sites (Fallout engine also uses acm format, so these guys have all needed utilities)

http://potenciusnet.teambg.net/

http://www.fallout.ru/teamx/


Firstly we have to increase the speed of chosen wav sound by 100% (all wac sounds have increased speed)
Then we have to compress it using snd2acm to acm file (in the same time we note somewhere the output file size - this is our compressed acm file size)
Then we decompress it again using acm2wav in aim to determine the decompressed file size (amount of written bytes).

When we have these two numbers we can edit our acm file in hex editor : we add the header as described above and translate compressed and decompressed sizes to hexadecimal numbers and enter it as indicated in header description.
We can check if our file is valid using wavc converter - if the output is the same as source wav file then we have succeeded.

Then if you want to use these files in IE game you have to change extension of your wavc file to wav and put into override. and voila ! we have wavc compressed files.
Make sound reference in dialog.tlk if your wavc file serves as dialogue / speech sound.
