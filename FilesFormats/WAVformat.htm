WAV file format

FileExtension : WAV
Content : Description of WAV format
History : File created by Manveru
Info taken from MalatanarTeamBG's posts (25.11.2001)

 

WAV Files are used for player character sounds ("soundsets") also for FX sounds in game. They are in the /sounds folder under the game directory. Their naming conventions are documented in the game readme files. They must be 22050 Hz, 16 bit and mono.


WAV files can be created using various programs, and converted to ACM or WAVC.
