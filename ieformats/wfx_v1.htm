WFX file format

Applies to:
BG2, BG2: ToB

General Description
WFX files determine variation for playback of wave sounds. A WFX controls the variation for the similarly named WAV file (e.g. mySound.wfx controls the variation for mySound.wav).

Detailed Description

WFX files consist of only a header. Each WFX file has a constant length of 108h (264) bytes.

Offset Size (datatype) Description
0x0000 4 (char array) Signature ('WFX ')
0x0004 4 (char array) Version ('V1.0')
0x0008 4 (dword) Unknown, always 0
0x000c 4 (dword) Flags:
bit 0: Unknown
bit 1: Unknown
bit 2: Enabled
0x0010 4 (dword) Deviation.
0x002c 244 (bytes) Unknown, unused? Always 0.
