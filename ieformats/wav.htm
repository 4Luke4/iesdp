WAV file format

Applies to:
BG1, BG1: TotS, BG2, BG2: ToB, PST, IWD, IWD:HoW, IWD:TotL, IWD2


General Description
WAV Files are used for player character sounds ("soundsets") also for FX sounds in game. They are in the /sounds folder under the game directory. The naming conventions are documented in the game readme files. They must be 22050 Hz, 16 bit and mono.
WAV files can be created using various programs, and converted to ACM or WAVC.
