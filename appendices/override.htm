Override Folders

Several folders in the main game directory can be used as override folders. The following folders functions have this function, and are prioritised in the listed order:
  1. Characters (overrides all others)
  2. Portrait
  3. Sounds
  4. Scripts
  5. Override (overriden by all others)


Baldur's Gate Enhanced Edition adds additional override folders located in the user profile directory:
  1. GAME\Movies (overrides all others)
  2. GAME\Lang\XYZ\Movies
  3. USER\Characters
  4. USER\Portraits
  5. GAME\Sounds
  6. GAME\Lang\XYZ\Sounds
  7. GAME\Scripts
  8. GAME\Override (is overriden by all others)
The lang folder used by the engine is dicated by the current language.
