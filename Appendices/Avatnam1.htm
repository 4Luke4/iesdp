Standard Character Avatars naming convention


This is very useful for someone who wants to edit a characters appearance or animation. 
Example: if id like to change the walking animation of human male cleric id have to look each .bam file and it could take forever, that's way i came up with this "technique", so u can find the right animation.


1.=character (irrelavent)

2.=race - 

E=elf,half-elf. 
h=human. 
D=dwarf,gnome. 
I=halfling

3.=gender - 

F=female, 
M=male
4.=class - 
W=mage. 
C=cleric. 
F=fighter,ranger,paladin. 
T=thief,bard.

5.=armor - 

1=no armor. 
2=leather. 
3=robe. 
4=platemail,mail

6.=action - 

a=attak. 
c=cast spell. 
g=movement, standing, sleeping, being dead etc. 
s=shooting. 
w=walking.

7.=if 6.=a then -  

1=single-handed weapon overhead swing 
2=two-handed weapon overhead swing 
3=single-handed weapon backhand slash 
4=two-handed weapon backhand slash 
5=single-handed weapon thrusting 
6=two-handed weapon thrusting


7.=if 6.=s then - (empty)=bow. x=crossbow.
If there is an E in the end of the file, it means that it's the missing part of the
file without an E.

Example:CIMT1CA means: a male halfling with no armor is casting a spell.
Another example:CHMF4A5 means: a female human with platemail is attacking with a dagger.
